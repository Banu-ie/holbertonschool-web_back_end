# Python Async Function

This project covers asynchronous programming in Python using asyncio.

## Files
- `0-basic_async_syntax.py` - Basic async coroutine
- `1-concurrent_coroutines.py` - Running multiple coroutines concurrently
- `2-measure_runtime.py` - Measuring async runtime
- `3-tasks.py` - Creating asyncio Tasks
- `4-tasks.py` - Task-based concurrent coroutines
